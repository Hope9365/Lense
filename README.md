<h1 align="center">Lense</h1>

<p align="center">
  <strong>Lense</strong> is a QoL utility mod for Hytale. It displays a tooltip with information about the block or entity you are currently looking at.
</p>

---

## 🚧 Early Access 🚧

**Please Note:** Just like Hytale itself, Lense is currently in **Early Access**.

This means the mod is still actively being developed and will receive **significant updates and changes** over time.

At the moment, only **a part of the planned features/content** is implemented — more functionality is coming in future updates.

Bugs may occur, and feedback is highly appreciated!

## 🛠️ Features

### ⚙️ Per-player config
*   Configuration is unique to each player, allowing for a personalized experience.

### 🌱 Blocks HUD
*   Block Name
*   Mod Name
*   Block Breaking Progress Bar
*   Containers (Chests) Inventory
*   Furnace Processing Progress Bar
*   Bench Tiers
*   Farming Growth Percent
*   Teleporter Warps Info

### 👤 Entity HUD
*   Entity Name
*   Mod Name
*   HP

## 💻 Commands

*   `/lense` - The main command for the mod.
*   `/lense toggle` - Toggles the HUD visibility (Show/Hide).
*   `/lense toggle on` - Explicitly enables the HUD.
*   `/lense toggle off` - Explicitly disables the HUD.
